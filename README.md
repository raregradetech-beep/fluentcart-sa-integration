# FluentCart SA Integration  
**South-African e-commerce super-pack for FluentCart**

## What you get today
- ✅ PayFast gateway (redirect + ITN)  
- ✅ Meta WhatsApp Cloud-API client (text messages)  
- ✅ Automatic “payment confirmed” WhatsApp on order-paid

## Quick test
1. Install FluentCart (free) on a staging site.  
2. Upload this plugin folder to `wp-content/plugins/` and activate.  
3. Enter **sandbox** credentials:  
   - PayFast → Settings → Sandbox = ON, Merchant ID & Key from sandbox dashboard.  
   - WhatsApp → Settings → Meta access-token & phone-number ID.  
4. Store phone number must be **verified** in your Meta app.  
5. Place an order, choose PayFast, pay with card `4111111111111111` (any future expiry, any CVV).  
6. You should receive a WhatsApp within seconds.
